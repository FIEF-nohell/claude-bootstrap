# Archived versions

Past versions of `claude-bootstrap-prompt.md` from the repo root.

Filename format: `claude-bootstrap-prompt-v<N>.md` where `N` increments by 1 on every change. Lowest number is the oldest archive, highest number is the most recently superseded version. The current working version is always in the repo root, not here.

Do not edit files in this folder. They are immutable history.
